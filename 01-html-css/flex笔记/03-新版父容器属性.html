基于display:flex新版父容器属性（支持安卓4.4+,iOS 9.0+, iOS7.0+ 需添加-webkit前缀）

    
        .flex-box {
            display: -webkit-flex;
            display: flex;

            flex-direction: row 丨 row-reverse 丨 column 丨 column-reverse;
            /* 设置子元素主轴方向：横向从左到右(默认) 丨 横向从右到左 丨 纵向从上到下 丨 纵向从下到上 */

            flex-wrap: nowrap 丨 wrap 丨 wrap-reverse;
            /* 子元素过多时是否换行: 不换行(默认) 丨 换行 丨 换行并第一行在下方 */

            flex-flow: <flex-direction> 丨丨 <flex-wrap>
            /* 主轴方向与换行简写 */

            justify-content: flex-start 丨 flex-end 丨 center 丨 space-between 丨 space-round;
            /* 设置子元素在主轴方向上的对齐方式：左对齐（默认） 丨 右对齐 丨 居中对齐 丨 两端对齐 丨 平均分布 */

            align-items: flex-start 丨 flex-end 丨 center 丨 baseline 丨 stretch;
            /* 设置子元素在交叉轴方向上的对齐方式：顶部对齐（默认） 丨 底部对齐 丨 居中对齐 丨 上下对齐并铺满 丨 文本基线对齐 */

            align-content: flex-start 丨 flex-end 丨 center 丨 space-between 丨 stretch;
            /* 多行时子元素在主轴方向上的对齐方式：顶部对齐 丨 底部对齐 丨 居中对齐 丨 上下对齐并铺满 丨 上下平均分布（默认） */
        }

1.flex-direction 设置子元素主轴方向

1.1 flex-direction:row; 横向从左至右排列（默认）

对应旧版语法为-webkit-box-orient:horizontal; -webkit-box-direction: normal;

1.2 flex-direction:row-reverse; 横向从右至左排列

对应的旧版语法为box-lines,但是几乎所有浏览器都不支持，慎用。当元素过多时，不会换行，但是宽度会被压缩。

3. justify-content 设置子元素在主轴方向（水平方向）上的对齐方式

3.1 justify-content: flex-start;设置子元素在主轴上左对齐(默认)

对应的旧版语法为-webkit-box-pack: start;

1
2
3

3.2 justify-content: flex-end;设置子元素在主轴上右对齐

对应的旧版语法为-webkit-box-pack: end;

1
2
3

3.3 justify-content: center;设置子元素在主轴上居中对齐

对应的旧版语法为-webkit-box-pack: center;

1
2
3

3.4 justify-content: space-between;设置子元素在主轴上两端对齐

对应的旧版语法为-webkit-box-pack: justify;

1
2
3

3.4 justify-content: space-around;设置子元素在主轴上平均分布

无对应旧版语法，慎用，可能某些低版本安卓/iOS系统不支持
